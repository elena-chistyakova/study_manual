# Установка и настройка утилиты распознавания

Комплект поставки утилиты распознавания удостоверений личности следующий:

*	Утилита *Passport Reader* (включая клиентскую часть, службу сканирования и необходимые конфигурационные файлы).
*	Дистрибутив программного продукта *Content AI PassportReader SDK*.
*	Лицензия (лицензии) на программный продукт *Content AI PassportReader SDK* (серийные номера и файл *LicensingSettings.xml*).
* 	Инструкции по активации лицензии отдельным файлом.
